# Conversion Path Analysis

## Overview
Conversion path analysis studies how users move through a website
before completing a purchase or lead form.

## Best Fit
Recommended when traffic remains stable but conversions decline.

## Deliverables
- funnel analysis
- user behavior review
- usability recommendations
- friction point identification

## Inputs Required
Analytics data
Conversion tracking configuration

## Typical Findings
Confusing navigation, slow checkout pages,
or unclear calls to action.

## Exclusions
Does not include full website redesign.

## Prerequisites
Accurate analytics and conversion tracking.

## Success Criteria
Improved conversion rates and smoother user journeys.

## Recommended Pairings
Often paired with on‑page optimization.

## Notes
Understanding user behavior helps improve website performance.
