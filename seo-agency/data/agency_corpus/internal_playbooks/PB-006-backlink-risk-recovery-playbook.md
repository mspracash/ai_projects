# Backlink Risk Recovery Playbook

## Trigger
Use this playbook when a client suspects harmful backlinks affecting search rankings.
Symptoms include ranking volatility or warnings from Search Console.

## Immediate Checks
Review backlink profile metrics using link analysis tools.
Check anchor text distribution for unnatural patterns.
Identify sudden spikes in low-quality links.

## Diagnostic Questions
Determine whether the client previously engaged in aggressive link-building campaigns.
Ask whether automated directory submissions or link networks were used.

## Evidence Sources
Backlink analysis tools reveal domain quality and link sources.
Search Console link reports show inbound link trends.
Spam scoring tools highlight suspicious domains.

## Common Root Causes
Link networks generating artificial backlinks.
Low-quality directory listings.
Negative SEO attempts from competitors.

## Recommended Services
Backlink audit.
Toxic link identification.
Disavow file preparation.
Ongoing link monitoring.

## Execution Pattern
Classify backlinks by quality.
Compile toxic domains into disavow file.
Submit disavow file through Search Console.

## Escalation Path
Severe cases may require extended monitoring or reconsideration requests if penalties occur.

## Example Signals
Large number of exact-match anchor links.
Backlinks from unrelated foreign domains.
Rapid spikes in new inbound links.

## Resolution Pattern
Search engines gradually discount harmful links after disavow submission.
Ranking stability usually improves over time.

## Notes
Regular backlink audits reduce long-term risk.
