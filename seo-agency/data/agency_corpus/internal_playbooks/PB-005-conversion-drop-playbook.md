# Conversion Drop Playbook

## Trigger
This playbook is activated when website traffic remains stable but conversions decline.
Examples include fewer purchases, lead submissions, or completed contact forms.

## Immediate Checks
Confirm conversion tracking accuracy in analytics systems.
Verify that forms and checkout processes function correctly.
Check whether recent design changes affected usability.

## Diagnostic Questions
Ask whether the client changed pricing, messaging, or landing page layouts.
Determine whether traffic sources have shifted toward lower-intent audiences.

## Evidence Sources
Analytics platforms reveal funnel drop-off points.
Heatmap tools show user interaction patterns.
Session recordings may highlight usability problems.

## Common Root Causes
Conversion declines often occur due to confusing page layouts.
Slow page speed can also discourage users.
Poor alignment between keywords and landing page content may reduce conversions.

## Recommended Services
Conversion path analysis.
Landing page optimization.
User experience improvements.
Performance optimization.

## Execution Pattern
Simplify conversion forms.
Clarify calls to action.
Improve page load times.
Align messaging with user intent.

## Escalation Path
Complex usability issues may require design specialists.
Major checkout failures require developer intervention.

## Example Signals
Stable traffic but reduced leads.
High bounce rates on landing pages.
Abandoned checkout processes.

## Resolution Pattern
Conversion improvements typically appear quickly once friction points are removed.

## Notes
Conversion analysis should focus on user behavior rather than traffic volume alone.
